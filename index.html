Politique de Confidentialité - Darts Games

Cette Politique de Confidentialité décrit comment l'application Darts Games collecte, utilise et protège les informations des utilisateurs. En utilisant l'application, vous acceptez les pratiques décrites dans cette politique.

Données collectées

Nous ne collectons aucune information personnelle identifiable, telles que nom, adresse e-mail ou numéro de téléphone.

L'application peut collecter des informations anonymes sur l'utilisation et les erreurs via des services tels que Firebase Analytics pour améliorer l'expérience utilisateur.

Utilisation des données

  • Améliorer la stabilité et les performances de l'application
  • Comprendre comment l'application est utilisée afin de proposer de nouvelles fonctionnalités

Partage des données

Nous ne partageons pas vos données personnelles avec des tiers.

Certaines données anonymisées peuvent être stockées sur les serveurs de Google via Firebase pour l'analyse statistique.

Sécurité

Nous prenons des mesures de sécurité raisonnables pour protéger vos informations contre l'accès, l'altération ou la divulgation non autorisée.

Droits de l’utilisateur

Les utilisateurs peuvent à tout moment demander la suppression des données collectées anonymement via l'application.

Modifications de la politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Les modifications seront publiées sur cette page.

Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter à : support@dartsgames.com

Privacy Policy - Darts Games

This Privacy Policy describes how the Darts Games app collects, uses, and protects user information. By using the app, you agree to the practices described in this policy.

Data Collection

We do not collect any personally identifiable information, such as name, email address, or phone number.

The app may collect anonymous usage and error information via services such as Firebase Analytics to improve the user experience.

Use of Data

  • Improve app stability and performance
  • Understand how the app is used to provide new features

Data Sharing

We do not share your personal data with third parties.

Some anonymized data may be stored on Google servers via Firebase for statistical analysis.

Security

We take reasonable security measures to protect your information from unauthorized access, alteration, or disclosure.

User Rights

Users can request the deletion of anonymously collected data at any time via the app.

Policy Updates

We may update this Privacy Policy from time to time. Changes will be posted on this page.

Contact
